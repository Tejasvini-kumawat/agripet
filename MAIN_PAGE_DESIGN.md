# AgriPet Main Page Design

## Design Concept
A modern, agricultural-themed landing page that's accessible to rural farmers while maintaining professional healthcare aesthetics.

## Color Palette
- **Primary Green**: #10b981 (agriculture, growth)
- **Secondary Blue**: #3b82f6 (healthcare, trust)
- **Accent Orange**: #f59e0b (emergency, alerts)
- **Neutral Grays**: #f3f4f6, #6b7280, #1f2937

## Mobile-First Design Features

### Responsive Breakpoints
- **Mobile**: < 768px (Stacked layout, hamburger menu)
- **Tablet**: 768px - 1024px (2-column grid)
- **Desktop**: > 1024px (Full grid layout)

### Accessibility Features
- **Large Touch Targets**: Minimum 44px buttons
- **High Contrast**: WCAG AA compliant colors
- **Voice Navigation**: Screen reader friendly
- **Multi-language Support**: Regional language toggle

### Interactive Elements
- **Floating Action Button**: Quick emergency access
- **Progress Indicators**: Upload/analysis progress
- **Toast Notifications**: Success/error messages
- **Modal Dialogs**: Appointment booking, image upload

- **Loading States**: Skeleton screens for image uploads
